# Robustness Experiments

80/20 held-out scenario splits (20 random seeds) used to assess whether the deployable layouts generalise to unseen ventilation states.

## Status
This family is planned / empty. No driver has been ported yet.

The robustness results and plots that appear in the paper were generated with the historical `robustness_analysis.py` from the BarnCSP checkout.

## Current practical path
- The published robustness numbers and any associated figures are in the Zenodo result archives.
- The clean GBM API + the documented seeds make it straightforward to re-run equivalent held-out evaluations on the full Zenodo scenario set when needed.

## Seeds
20 independent seeds (documented in the original robustness logs).
