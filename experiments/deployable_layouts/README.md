# Deployable Layouts (Fixed Sensor Placement)

This family produces the main result tables and heatmaps of the paper:
"Deployable-layout MAE for CO₂ and NH₃ at k=5,10,20 across LW ratios."

## What it does
1. For each method (GBM + 8 baselines), collect the best k-point positions from every scenario (per LW, dim, gas).
2. Pool them and run KMeans (fixed random_state=0) → one set of centroids = the single fixed deployable layout for that (gas, dim, LW, k).
3. For every scenario, evaluate the layout (nearest-grid lookup + mean) vs. the true barn mean.
4. NH₃ additionally reports plume-only MAE (plume = values > 1% of max concentration in the scenario).
5. Produces `deploy_results_all.json`, ranking tables, critical difference diagrams, etc.

## Key outputs (committed in ../results/aggregates/)
- `deploy_results_all.json`
- `ranking_tables.tex`
- PDFs: loss_by_k_and_lw.pdf, nh3_loss_by_k_and_lw.pdf, critical_difference_diagrams.pdf, etc. (regenerated by plot scripts)

## Reproduction steps (high level)
```bash
# From the repo root (after installing the package)
python -m experiments.deployable_layouts.run \
    --results-root ../results \
    --k 5,10,20 \
    --methods gbm,kmedoids,greedy,uniform,random,simulated-annealing,pso,monte-carlo,genetic \
    --out ../results/aggregates/deploy_results_all.json
```

See the family `run.sh` / Makefile (and any runner in the directory) for exact flags and seeds.

## Seeds & determinism
- KMeans: `random_state=0` (or 42 in some variants — documented in the script).
- All upstream per-scenario positions come from the single stored stochastic run per scenario (no multi-seed averaging).

## Data requirements
- Per-scenario JSONs from the other families (or the full Zenodo archive).
- A small number of representative scenarios are sufficient for smoke tests (see `../data/examples/`).

## Relation to the paper
This family directly generates:
- Table 1 (operational roles) context
- Figure 4 (NH₃ deployable MAE heatmap with callouts where baselines win)
- The CO₂ deployable heatmaps
- The ranking / CD diagrams in the main body and Appendix D

## Notes on gas dependence
GBM dominates for broad-plume CO₂. For NH₃ (sharp localised plumes) the best fixed layout is more gas- and k-dependent — uniform, random, greedy or PSO sometimes win on specific LW/k cells. This is a deliberate, balanced finding reported in the revision.

Full per-scenario JSONs + the complete deploy_results_all.json live in the Zenodo results archive (see top-level data/README.md for DOI).
